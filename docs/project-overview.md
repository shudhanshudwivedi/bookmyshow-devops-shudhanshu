# BookMyShow DevSecOps Project

## Project Overview

This project demonstrates the deployment of a BookMyShow clone application using a complete DevSecOps pipeline.

## Tools Used

- GitHub
- Jenkins
- SonarQube
- Trivy
- Docker
- Kubernetes

## CI/CD Workflow

1. Source Code stored in GitHub
2. Jenkins Pipeline triggered
3. SonarQube Code Analysis
4. Trivy Security Scan
5. Docker Image Build
6. Docker Image Push
7. Kubernetes Deployment
8. Kubernetes Service Exposure

## Project Outcome

Successfully automated application deployment using DevSecOps best practices.
